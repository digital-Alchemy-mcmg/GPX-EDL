# Envoy

Envoy owns the operational path from an accepted Disk Definition through sourcing, normalized observations, Zeta projection, and promotion preparation.

## Initial inputs

- Disk Definition
- source adapter configuration
- normalized listing observations

## Initial outputs

- Disk Cartridge
- inspectable Zeta output
- operator decision record
- promoted listing artifact

## Authority limits

Envoy may not infer approval, rewrite operator decisions, prepare candidate-facing submission artifacts, or submit applications.
